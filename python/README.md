# Simulador PSO & Algoritmo Evolutivo en Python

## Requisitos

- Python 3.8 o superior


## Configuración del entorno virtual
Este proyecto utiliza librerías como PyQt5 para UI y pygame para la simulación. Para instalar las dependencias y correr el proyecto es recomendable utilizar un entorno virtual.

### 1. Crear el entorno virtual

```bash
python -m venv venv
```

### 2. Activar el entorno virtual

**Windows:**
```bash
venv\Scripts\activate
```

**Mac / Linux:**
```bash
source venv/bin/activate
```

Sabrás que está activo porque el prompt cambia a algo como `(venv) C:\...`

### 3. Instalar dependencias

```bash
pip install -r requirements.txt
```

### 4. Ejecutar el simulador

```bash
python simulador.py
```

---

## Desactivar el entorno virtual

Cuando termines, puedes desactivarlo con:

```bash
deactivate
```
